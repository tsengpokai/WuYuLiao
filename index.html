數據模擬原理

  • 背景：高斯白雜訊（Gaussian noise）
  • 事件：類 Ricker 的突發能量（模擬 P/S 波）
  • Normalization：振幅壓到 -1~1（更穩定、收斂更快）

視覺化（plot_results 概念）

  • 波形 + 真實標籤 + AI 機率
  • 判對：標題綠色；判錯：紅色
  • 每次生成不同，但模型應學會區分
matplotlib-like UI simulation
進階應用：從分類到分割（Segmentation）

入門模型回答「整段有沒有事件」。進階模型會對每個時間點輸出類別（例如 P、S、Noise）， 以機率函數精準標記 P/S 到時（例如 U-Net 類、PhaseNet、EQTransformer 等）。

真實數據與工程落地方向

可用 obspy 讀 SAC/MSEED；結合地震目錄資料集訓練；實務上常輸入三分量（E/N/Z）或多測站資料提升可靠度。

MODULES

四大核心模組（漂亮卡片 + 專業解釋）

01

相位拾取

Phase Picking
RED-PAN

將原始波形轉成 P/S 機率函數，跨過門檻即產生 picks。本網站用「機率曲線 + threshold」視覺化 Phase Picking。

02

相位關聯

Phase Association
GaMMA

把多測站的 picks 連成同一個事件。本網站以「事件窗 + 多站一致性」示範，並用地圖連線呈現關聯結果。

03

震源定位

Location
NonLinLoc

用多站到時資訊估計震央位置。本網站用 2D grid-search（教學版）求出震央並畫誤差圈。

04

規模估計

Magnitude
Local Regression

以最大振幅與距離衰減估 ML。本網站用簡化回歸式示範（非正式地震學公式）。
